# Модули JARVIS

## Core

Ядро системы.

Отвечает за взаимодействие между всеми модулями.

---

## Memory

Долговременная память.

- сохранение информации;
- поиск воспоминаний;
- работа с pgvector.

---

## AI

Работа с LLM.

- OpenAI;
- обработка запросов;
- генерация ответов.

---

## Automation

Автоматизация процессов.

- n8n;
- сценарии;
- интеграции.

---

## Interfaces

Точки взаимодействия с пользователем.

На текущий момент:

- Telegram.

В дальнейшем:

- Web;
- Voice;
- API.

---

## Storage

Хранилища данных.

На текущий момент:

- PostgreSQL.

В дальнейшем возможно:

- Redis;
- S3;
- Qdrant.
