## Именование

### Папки

Используется нижний регистр.

Примеры:

- docs
- prompts
- automation
- storage

---

### Файлы

Используется UPPER_SNAKE_CASE для документации.

Примеры:

- README.md
- ROADMAP.md
- STACK.md
- CHANGE_POLICY.md

Для остальных файлов используется snake_case.

Примеры:

- long_term_memory.json
- recall_memory.md
- user_memory.sql
