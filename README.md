# JARVIS

Модульная AI-платформа для создания персональных и корпоративных AI-ассистентов, автоматизации бизнес-процессов и интеграции больших языковых моделей (LLM) в реальные задачи.

---

## Возможности

На текущем этапе реализуются:

- долговременная память;
- AI-агенты;
- интеграция с OpenAI;
- автоматизация через n8n;
- Telegram-бот;
- модульная архитектура;
- развертывание в Docker.

---

## Стек технологий

- OpenAI API
- n8n
- PostgreSQL + pgvector
- JavaScript
- Docker
- Linux
- Git

## Структура проекта

```text
JARVIS/

├── architecture/   # Архитектура проекта
├── assets/         # Изображения и ресурсы
├── automation/     # Автоматизация и workflow
├── docker/         # Docker-конфигурации
├── docs/           # Документация
├── prompts/        # AI-промпты
├── scripts/        # Вспомогательные скрипты
├── src/            # Исходный код
├── storage/        # Работа с хранилищами данных
└── tests/          # Тесты
```
